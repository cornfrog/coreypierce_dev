## 🛠️ Built With
This site is powered by:
- **Framework:** [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** [Material UI (MUI)](https://mui.com/), [Emotion](https://emotion.sh/), & [Styled Components](https://styled-components.com/)
- **Routing:** [React Router 7](https://reactrouter.com/)
- **Interactivity:** [Embla Carousel](https://www.embla-carousel.com/) for smooth sliders
- **Communication:** [EmailJS](https://www.emailjs.com/) for the contact form

## 🏃 Local Development
If you want to take a look under the hood locally:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/cornfrog/coreypierce_dev.git
   ```
2. **Install the goods:**
   ```bash
   npm install
   ```
3. **Launch the rocket:**
   ```bash
   npm run dev
   ```
